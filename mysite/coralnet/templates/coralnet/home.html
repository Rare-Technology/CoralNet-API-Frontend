CoralNet Deploy API Web

Welcome to the web version of Scott Miller's CoralNet API Deployer. This website is meant to abstract the need to create a Dropbox account or to touch any code. Simply provide the following information:

  1. The number of images you will upload. It is important you get this number correct so that we know when your images are done uploading.
  2. Your CoralNet model's global id:
  3. An email address to send the results to.
  4. The images you want annotated. You will be prompted to upload images to Dropbox after submitting the form below.
